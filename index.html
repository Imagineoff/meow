Majk AI Bot

About Majk

Majk is a Discord AI bot powered by OpenAI (GPT-4 style) that can interact with users in a friendly or aggressive manner depending on its personality settings. You can customize the bot's personality, set up a dedicated channel for replies, and enjoy robust AI-powered interactions.

Status: Online | Servers: 0

Commands

/setupchannel
Set the channel where Majk will respond.
/uninstallchannel
Remove the AI reply channel.
/personality
Set or update Majk's personality (admin only).
/resetpersonality
Reset Majk's personality to default (admin only).
/help
Show available commands.
Owner Commands
Manage servers, restart, shutdown, view logs.

Features

- Responds to messages only in the configured channel.

- Stores chat history for coherent AI replies.

- Fully admin-controlled personality and channel settings.

- Owner-only commands for blacklist, invite creation, and bot management.

- Dark/Light mode toggle for easier viewing.
